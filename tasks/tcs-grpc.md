# Подключить grpc-клиенты в проекте br-tcs для запросов к другим проектам

В br-tcs надо добавить клиенты grpc для подключения к:
- br-users
- br-accounts
- br-billing
- br-trading
- br-kyc

Их апи находится в библиотеке br-contracts которая уже подключена в данном проекте.
Тебе надо запилить их сборку в DI (как в других проектах),
а также пробросить эти клиенты в notificationtemplatedata.
