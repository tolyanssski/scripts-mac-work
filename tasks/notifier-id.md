# Правки в проект br-users, пакет notifier

Найди где работает кафка-воркер, принимающий сообщения из топика private-messages.
Там принимается dto-сообщение конкретному юзеру, и на основе сообщения создается
Notification.
Нужно в dto добавить опциональное nillable поле NotificationID (uuid),
и если оно пришло заполненное, то создавать новый Notification уже сразу с этим
предопределенным id.
А если не заполнено, то оставить старую логику (то есть генерировать новый id).
