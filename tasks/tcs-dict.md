# Собрать все используемые eventType в проектах где отправляется событие в TCS

Пройдись по всем проектам в текущей папке, конкретно по проектам названия которых начинаются
с br-.
Найди места где формируются и отправляются в кафку события в TCS-топик.
Там есть отправляются сообщения с полем EventType, само значение везде захардкожено.
Нужно собрать все используемые значения EventType, и затем использовать их в следующей 
части этой задачи.

# Новая сущность в проекте br-tcs - справочник значений EventTypes

Создай новую сущность внутри проекта br-tcs и миграцию.
В сущности поля:
- id (uuid)
- event_type (string)
- created_at
- updated_at

В миграции запили не только создание таблицы, но и первичное наполнение
строками с вариантами event_type, которые ты нашел в предыдущей части задачи.
Миграция должна быть внутри проекта br-tcs.
